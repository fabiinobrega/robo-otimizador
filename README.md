# 🤖 Robô Otimizador - Sistema de Automação de Marketing

Sistema completo de automação de marketing com integrações reais para OpenAI, Manus AI, Facebook Ads e Google Ads.

## 🚀 Funcionalidades

- ✅ **Gerador de Copy com IA** - OpenAI e Manus AI
- ✅ **Análise de Concorrentes** - Scraping e análise real
- ✅ **Facebook Ads** - Criação de campanhas reais
- ✅ **Google Ads** - Configuração completa
- ✅ **Upload de Mídia** - Suporte a imagens e vídeos
- ✅ **Dashboard** - Métricas e status das APIs

## 🔧 Configuração

1. Clone o repositório
2. Configure as variáveis de ambiente (use `.env.example` como modelo)
3. Deploy no Render

## 🔑 Login Padrão

- **Usuário:** admin
- **Senha:** admin123

## 📱 APIs Suportadas

- **OpenAI** - Geração de copy com GPT
- **Manus AI** - IA alternativa para copy
- **Facebook Ads** - Graph API v18.0
- **Google Ads** - API completa

## 🌐 Deploy no Render

1. Conecte seu repositório GitHub
2. Configure as variáveis de ambiente no painel do Render
3. Deploy automático via Dockerfile

## 📋 Variáveis de Ambiente

Veja `.env.example` para a lista completa de variáveis necessárias.
