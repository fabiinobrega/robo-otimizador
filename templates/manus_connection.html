Configuração
Como Configurar
  1. Entre em contato com o suporte Manus em support@manus.im
  2. Solicite acesso à API oficial e credenciais OAuth2
  3. Configure as variáveis de ambiente:
    • MANUS_API_BASE_URL
    • MANUS_CLIENT_ID
    • MANUS_CLIENT_SECRET
  4. Clique em "Conectar Agora" para autorizar
Importante

Esta integração requer credenciais oficiais da API Manus. Sem as credenciais, o sistema continuará funcionando normalmente, mas sem sincronização com a plataforma Manus.
